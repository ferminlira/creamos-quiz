# Project Overview

## Purpose
A 20-question trivia game about Creamos' portfolio, awards, and history.
Played synchronously by remote team members during AI Fridays sessions.

## Live URL
https://creamos-quiz.vercel.app

## Access
Password protected: username `admin`, password `Creamos.123456`

## Gameplay
- Players join a lobby, anyone can start the game
- 15 seconds per question, all players see questions simultaneously
- 100 points per correct answer
- Final leaderboard shown at the end

## Branding
- Font: Poppins (Google Fonts)
- Primary color: #F5E600 (yellow)
- Background: #080808 (near black)
- Style reference: creamosagencia.com
